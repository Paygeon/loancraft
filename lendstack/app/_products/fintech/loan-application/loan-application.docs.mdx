<HowToStep index='1' title="Add script before the </head> tag">


<CodeBlock fileName="index.html">

```html
<script src="vashong.github.io/Loan-Widget/dist/loan-widget.js"></script>
```

</CodeBlock>

<br/>
Loan-Widget by Finance Chest - Follow instructions by [finchest - loan-widget](https://www.npmjs.com/package/loan-frontend) for how to input the configurations.

<CodeBlock fileName="index.html">
```html
<script>
var handler = LoanWidgetLibrary.create({
        partnerId: 'partner1',
        themeColor: '#007bff',
        partnerLogo: './logo192.png',
        fontColor: '#784523',
        typography: 'Arial, sans-serif',
        mode: 'light',
      });
      handler.initializeButton("#start");
</script>
```
</CodeBlock>

<br/>
Add the html code anywhere in your website or app you want to display the button at.
<CodeBlock fileName="index.html">
```html
 <button id="start">Open Loan Application Modal</button>
```
</CodeBlock>

</HowToStep>
